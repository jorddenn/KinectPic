# KinectPic


Usage:
1. Install Zadig 2.1.0 https://github.com/pbatard/libwdi/releases/download/b658/zadig-2.1.0.exe
The  ost recent version will give an error
2. Connect Kinect
3. Open Zadig -> Options -> List All Devcies
4. Select Microsoft Kinect Camera from the drop down
5. Beside driver select libusbK (v3.0.6.0) and install
6. Install Processing https://processing.org/
7. Open KinectPic
8. In Sketch -> Import Library -> Add Library -> search "kinect" -> install "Open Kinect for Processing"
9. Run sketch
10.Click anywhere to take a picture
11."image.jpg" and "depth.csv" are saved in the sketch directory
